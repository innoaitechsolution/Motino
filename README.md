# Motino

**Motino** is a small, responsive web app for a daily ritual: spin a rainbow wheel once per day and get a motivational quote. Quotes can be shared or copied, and your spin is remembered locally until the next calendar day.

## Features

- **Daily spin** — One spin per day, enforced with `localStorage` (with sensible recovery if stored data is invalid).
- **Rainbow wheel** — Eight-color conic gradient; only the outer disc spins so the center label and pointer stay fixed and readable.
- **Quote card** — Frosted card with share support via the **Web Share API**, with **clipboard** fallback when sharing isn’t available.
- **Responsive layout** — Polished on small and large screens; respects **reduced motion** preferences for animations.
- **Production-ready** — Vite build output and **Netlify** config (`netlify.toml`) for static hosting.

## Tech stack

| Area        | Choice                                      |
| ----------- | ------------------------------------------- |
| UI          | React 18                                    |
| Tooling     | Vite 5, `@vitejs/plugin-react`              |
| Styling     | Plain CSS (layout + wheel stylesheets)      |
| Typography  | Google Fonts — Inter, Poppins               |
| Data        | Static JSON (`src/assets/quotes.json`)      |

## Setup

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Run the dev server**

   ```bash
   npm run dev
   ```

   Open the URL Vite prints (typically [http://localhost:5173](http://localhost:5173)).

## Build

Create an optimized production bundle in `dist/`:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

For **Netlify**, connect the repo or deploy the `dist` folder; the included `netlify.toml` sets `publish = "dist"` and `npm run build` as the build command.
